# edu-system

基于Maven搭建的SSM框架的教育后端管理系统

# 后端技术选型

1.dao：mybatis

2.service：spring

3.view and controller：spring mvc

# 前端技术选型

1.vue框架

2.node.js

3.axios

# 开发工具

1.后端: IDEA 2019

2.前端: VS code

3.数据库客户端工具: SQLYog、Navicat

# 开发环境

1.JDK 11

2.Maven 3.6.3

3.MySQL 5.7
